#define ALIGN (   x)    CRYPTO_ALIGN(x)

Declare variable alignment of x bytes.

Used during the declaration of a variable, indicates to the device compiler that a variable should be aligned to the byte value represented by x.
For example, to declare a 32 byte aligned integer constant:

ALIGN(32) const int aligned_integer = 123;

xbyte value to align variable to
